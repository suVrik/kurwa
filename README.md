# kurwa

kurwa is a framework for making games.

Currently supported platforms and compilers:

Platform | Compiler
---------|--------------------------------------
Windows  | Microsoft Visual Studio 15 and higher
Linux    | Clang 6.0 and higher
Mac OS   | Clang 6.0 and higher
